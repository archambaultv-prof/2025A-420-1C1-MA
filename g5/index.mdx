# 1C1 - Exploration du domaine de l'informatique

Bienvenue sur le site du cours 1C1 - Exploration du domaine de l'informatique du [Collège de
Maisonneuve](https://www.cmaisonneuve.qc.ca/). Ce site présente le contenu du cours dispensé
par le professeur [Vincent Archambault-Bouffard](https://github.com/archambaultv-prof) durant la session d'automne 2025.

## Organisation du site

Le site est organisé en trois sections principales :

- **Échéancier** : Le calendrier détaillé du cours, semaine par semaine. Attention à consulter
  le *bon* groupe.
- **Notes de cours** : Le contenu théorique et les exercices pratiques
- **Évaluations** : Les informations relatives aux évaluations du cours

Toutes les sections sont accessibles via le menu de navigation situé en haut de la page.

## Communication
