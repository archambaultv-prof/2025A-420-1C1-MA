Les sites web que vous visitez tous les jours sont construits avec trois
technologies principales : **HTML**, **CSS** et **JavaScript**. Comprendre ces
trois langages est essentiel pour tout développeur web. Dans le cadre de ce cours,
nous allons nous concentrer sur **HTML** et **CSS**.


## Qu'est-ce que HTML ?

**HTML** (HyperText Markup Language) est le langage qui permet de créer la
structure d'une page web. Imaginez HTML comme la **charpente d'une maison** :
il définit où vont les murs, les portes, les fenêtres et les pièces.

Avec HTML, vous pouvez créer :
- Des titres et des paragraphes
- Des listes et des tableaux
- Des liens vers d'autres pages
- Des images et des vidéos
- Des formulaires

## Qu'est-ce que CSS ?

**CSS** (Cascading Style Sheets) est le langage qui rend votre page web belle
et attrayante. Si HTML est la charpente de la maison, CSS est la **décoration
et la peinture**. Il contrôle l'apparence visuelle de votre site.

Avec CSS, vous pouvez modifier :
- Les couleurs du texte et du fond
- La taille et le style des polices
- L'espacement entre les éléments
- La position des éléments sur la page
- Les animations et effets visuels

## Pourquoi séparer HTML et CSS ?

Cette séparation est importante car elle permet de :
- **Organiser** votre code de façon claire
- **Modifier** l'apparence sans toucher au contenu
- **Réutiliser** les styles sur plusieurs pages
- **Collaborer** plus facilement en équipe

## Exemple simple

Voici une page web basique pour vous donner une idée :

```html
<!DOCTYPE html>
<html>
<head>
    <title>Ma première page</title>
    <style>
        h1 { color: blue; }
        p { font-size: 18px; }
    </style>
</head>
<body>
    <h1>Bonjour le monde!</h1>
    <p>Ceci est ma première page web.</p>
</body>
</html>
```

Dans cet exemple :
- **HTML** crée un titre (`<h1>`) et un paragraphe (`<p>`)
- **CSS** rend le titre bleu et agrandit le texte du paragraphe. Ici, le code CSS est inclus
  dans la balise `<style>` à l'intérieur de `<head>`, mais il peut aussi être dans
  un fichier séparé.
