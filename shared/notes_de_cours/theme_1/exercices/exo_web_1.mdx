
## 🎯 Objectif pédagogique

- Utiliser VS Code
- Créer une page web simple

## 📜 Énoncé

### Étape 1

- Créez un nouveau dossier pour votre projet web.
- Ouvrez ce dossier dans VS Code.
- Créez un fichier `index.html` avec le code de base HTML.
- Ajoutez un titre et un paragraphe dans le corps de la page.
- Ouvrez `index.html` dans un navigateur web pour voir votre page.

### Étape 2

- Ajoutez du code CSS pour styliser votre page.
- Modifiez les couleurs, les polices et la mise en page.
- Utiliser un style différent pour le titre et le paragraphe.
- Enregistrez vos modifications et actualisez la page dans le navigateur.

### Étape 3

- Ajouter une image à votre page web.
    - Créez un dossier `images` dans votre projet et placez-y une image trouvée
      en ligne ou une image de votre choix.
    - Utilisez la balise `<img>` pour afficher l'image dans votre page.

### Étape 4

- Expérimentez avec d'autres éléments HTML comme les listes, les liens et les
  tableaux.
